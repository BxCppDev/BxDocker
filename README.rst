====================================================================
BxDocker - Helper scripts to build Bayeux 3 docker images 
====================================================================


Supported systems:

* Ubuntu Linux 22.04
* Ubuntu Linux 24.04



Build Docker images for Ubuntu 22.04
===================================================
  

.. code:: shell
	  
   $ cd ubuntu22.04/
   $ bash make_images.bash
   $ docker image ls
..




Build Docker images for Ubuntu 24.04
===================================================
  

.. code:: shell
	  
   $ cd ubuntu24.04/
   $ bash make_images.bash
   $ docker image ls
..


.. end
